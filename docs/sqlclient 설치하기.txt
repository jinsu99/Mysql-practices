==== sqlclient 설치하기

방법 (1) setting - project - python interpreter - + 눌러서 - mysqlclient 설치하기
방법 (2) terminal에 pip install mysqlclient 입력
확인하기 : setting - project - python interpreter


==== 설치 시 발생할 수 있는 에러 잡기
0. 반드시 Error 내용 확인 선행
0. python interpreter에 설치 여부 확인
> setting - project - python interpreter
 
1. sqlclient 버전 선택해 설치
> pip install mysqlclient == 1.3.12

2. pip 버전 확인
3. Python Interpreter 버전 확인
> setting - project - python interpreter - 상단부분
